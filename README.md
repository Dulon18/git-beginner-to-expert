# Git in 5 Days: Beginner to Expert

Learn Git from absolute beginner to advanced level in 5 days.

## Curriculum

- Day 1: Git Basics
- Day 2: Branching & Merging
- Day 3: GitHub Collaboration
- Day 4: Advanced Git
- Day 5: Expert Git

## Prerequisites

- Computer
- Terminal
- Git Installed

## Learning Outcome

After completing this course, you will:

- Understand Git deeply
- Collaborate using GitHub
- Resolve conflicts
- Recover lost work
- Use advanced Git workflows
- Contribute to open source projects
